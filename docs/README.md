# Clover 文档入口

`docs/` 现在收敛为两组文档：

- `architecture/`：面向当前仓库现实、问题和演进路线的系统级知识库
- `foundation/`：面向长期规范和设计基线的基础文档

阅读建议：

1. 先看 `architecture/README.md`
2. 再看 `foundation/README.md`

如果你是维护者，同时看：

- `../agent.md`
